# Katsina Online Business (KOB)

Katsina Online Business (KOB) dandali ne na kasuwanci da aka gina domin tallafawa ‘yan kasuwa a Katsina State ta hanyar haɗa kasuwanci, fasaha, da amincewa.

## 🚀 Abubuwan da Shafin Ke Yi
- Tallata kayayyaki da ayyuka
- Haɗa masu siye da masu sayarwa
- Rage scam ta hanyar ka’idoji da verification
- Tallafa wa kasuwancin zamani (online & offline)

## 🛠️ Fasahar da Aka Yi Amfani Da Ita
- HTML
- CSS
- JavaScript (Vanilla JS)
- Firebase (Auth + Firestore)
- GitHub Pages (Free Hosting)

## React (kob-react) development

The React app lives in the `kob-react/` folder. To run locally:

1. Copy `kob-react/.env.example` to `kob-react/.env` and populate the VITE_* vars (Cloudinary & Firebase)
2. cd `kob-react` and run `npm install`
3. `npm run dev` to start the dev server

Build for production with `npm run build`. Deploy the `dist/` folder to GitHub Pages or any static host.
